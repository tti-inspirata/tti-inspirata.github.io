常见问题

账号相关
  1. 1. 恩道电子书账号与Adobe ID是否相同？

    2. 恩道电子书账号在恩道平台注册，用于登录恩道电子书平台/恩道书房，Adobe ID在Adobe官方网站注册，用于使用Adobe的各项服务；但不排除您使用同一邮箱分别注册了恩道电子书账号与Adobe ID的情况。

  2. 2. 我在恩道电子书平台购书，为何需要注册Adobe ID？

    3. 恩道电子书均受Adobe DRM保护，您需要通过Adobe ID及密码登录ADE阅读软件，以此激活您所使用的设备，方可正常阅读受保护的电子书。

  3. 3. 如果忘记了Adobe ID的密码怎么办？

    4. 请点击这里进入Adobe ID的官方登录界面，并点击“忘记了密码”，根据提示重置密码。

  4. 4. 恩道电子书账号是否需要绑定一个Adobe ID？

    5. 目前我们尚未有此计划。但您购买的同一本书，若需在不同设备上阅读，只能通过同一个Adobe ID激活该设备。

    阅读软件相关
    1. 1. 为什么点击“下载本书”，会下载名为“URLLink.acsm”的文件？

      2. “URLLink.acsm”并非电子书本身，而是所占空间极小的描述符文件（descriptor file ），包含让阅读软件安全下载电子书的相关指令；电子书下载完成后即可删除此文件。

    2. 2. 如果在不同设备上安装了ADE阅读软件，且其中一台设备已成功添加从恩道购买的电子书，其他设备是否需要重复下载和添加的流程？

      3. ADE阅读软件支持多设备同步，如果其中一台设备已成功添加从恩道购买的电子书，该书会被自动同步至其他设备的ADE阅读软件中（但需确认其他设备也已通过同一个Adobe ID被激活）。

    3. 3. 阅读同一本电子书，有设备数量的限制吗？

      4. 同一个Adobe ID最多可同时激活六台设备，也就是说，同一本电子书最多可同时在六台设备上阅读，若超过此数量限制，请联系客服重置您的激活信息。

    4. 4. 为什么有的设备已被Adobe ID激活，但并未见到电子书自动同步？

      5. 网络可能会有延迟，请您稍作等候，并请确保用来激活设备的是同一个Adobe ID。

    5. 5. 如果在一台设备中已成功添加的电子书被删除，是否需要重新下载？

      6. 是的，您需要在这台设备中重新下载并添加此书，但若其他设备中也已下载了此书，则不会被自动删除。

    6. 6. 如果已下载并添加的电子书有更新，它会在ADE当中自动更新吗？

      7. ADE尚不支持电子书的自动更新，但恩道电子书平台会通过邮件提醒您图书的更新信息，您需通过如下操作将其更新到最新版本：①在ADE阅读软件中删除此书；②完全关闭ADE阅读软件；③重新打开ADE阅读软件，被删除的图书更新版将重新出现。

    7. 7. 在不同设备的ADE阅读软件中，所做的批注、书签、高亮等是否可以同步保存并更新？

      8. ADE暂不支持以上信息的同步保存及更新，敬请谅解。

    8. 8. ADE是否支持添加本地图书或文档？

      9. 是的，如果您需要在您的设备中打开PDF或ePub格式的文档，均可使用ADE阅读软件。

    9. 9. 我能使用Kindle阅读器来阅读从恩道电子书平台购买的图书吗？

      10. Kindle用户只能阅读在亚马逊平台上购买的电子书，但除了ADE阅读软件，恩道电子书的确还能够在其他支持Adobe DRM的平台或设备上阅读，请点击这里查看清单。

    10. 10. 恩道电子书平台为什么采用ADE，未来会开发自己的阅读器吗？

      11. 作为主内的专业出版机构，我们将版权保护置于优先考虑的问题，因此采用Adobe DRM保护技术，及与之相匹配的ADE阅读软件。若未来技术发展更加成熟，我们也会考虑在电子书版权能够得到保障的前提下自行开发阅读器，请弟兄姐妹们为这项事工代祷！
