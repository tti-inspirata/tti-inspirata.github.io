圖書購買相關

  1. 1. 如何購買電子書？

    2. （1）網頁端購書網址：ebook.endao.co；

    （2）安卓端App支持應用內購書，點擊這裡下載安裝。

    注意：通過App Store（蘋果應用商店）下載安裝的恩道電子書App僅可瀏覽書籍信息，暫不支持購書。

  2. 2. 購書時我可以選擇哪些支付方式？

    3. 恩道電子書支持支付寶及信用卡付款，支付寶付款目前僅面向中國大陸用戶。

  3. 3. 已購買的電子書是否可以申請退換/退款？

    4. 非常抱歉，電子書是數字閱讀產品，購買成功後不可申請退換或退款。

  4. 4. 如何閱讀已經購買的電子書？可以使用Kindle閱讀嗎？

    5. （1）您可以在iOS/安卓移動端、PC/Mac電腦端下載安裝恩道電子書App閱讀已購圖書，點擊這裡瞭解詳情。

    *電腦端閱讀器目前僅支持基礎閱讀和聽書，後續將逐步新增其他功能，敬請期待。

    （2）您無法通過kindle閱讀在恩道購買的電子書（Kindle用戶只能閱讀在亞馬遜平台上購買的電子書或不受DRM版權保護的電子書）。

  5. 5. 我購買的是什麼類型的電子書？

    6. 您購買的是恩道提供的EPUB格式、受DRM版權保護的電子書。

  6. 6. 我可以購買電子書送給朋友嗎？

    7. 在書籍詳情頁點擊「贈送」按鈕，即可購書贈友；付款成功後，您可以在「我的>贈書管理」中填寫受贈對象的郵箱，對方收到贈書兌換碼後可在其「我的>使用兌換碼」中兌換贈書。


圖書管理相關

  1. 1. App書架上為何看不到我購買的電子書？

    2. App書架僅展示您已經下載的書籍。您可以在App「我的>圖書」中找到已購買的電子書，點擊「加入書架」即可下載。

  2. 2. 如何管理書架中的電子書？

    3. 長按電子書封面或點擊書架右上角「編輯」圖標，即可進入書架管理模式，您可以對圖書進行批量移出或分組操作。「移出圖書」將刪除相應電子書的本地文件，但不會刪除您所做的筆記/划線。

  3. 3. 我將已經購買的電子書從書架移出後，如何繼續閱讀這本書？

    4. 從書架移出的電子書仍然會顯示在「我的>圖書」當中，您只需前往該頁面，重新將此書添加至書架即可。

  4. 4. 在App中更新電子書時，為什麼會提示「更新失敗」？

    5. 一般情況下，「更新失敗」均是網絡原因導致的，請檢查您的網絡連接或稍後重試。

  5. 5. 我可以在多台設備上用同一恩道賬號登入App並閱讀已經購買的電子書嗎？

    6. 同一恩道賬號最多可同時在5台設備中登入，超過5台設備將無法成功登入；也就是說，您最多可以在5台設備中閱讀已經購買的電子書；您可通過網頁端「我的>我的設備」來管理您綁定的設備。

  6. 6. 在一台設備中閱讀電子書時，我的閱讀/靈修進度、書架分組以及所做的筆記、划線、書簽，是否可以同步到其他設備當中？

    7. 可以的。您只需使用同一恩道賬號登入App，在網絡連接正常的情況下，以上數據即可自動同步更新；但在新的設備中，您仍需手動將這些電子書添加至書架。


圖書閱讀相關

  1. 1. 已經購買的電子書，可以離線閱讀嗎？

    2. 可以的，您只需將其添加至書架，電子書數據便已保存至本地；但離線閱讀時，若電子書有更新，則無法下載更新。

  2. 2. 如何在電子書中查看相應紙書頁碼？

    3. 點擊閱讀界面右上角的AA圖標，在彈出的菜單欄最下方，開啓「顯示紙書頁碼」，閱讀界面中出現的橙色標籤即為紙書頁碼（例如，標籤7至8之間的內容歸屬於紙書第7頁）。

  3. 3. 若遇到其他問題，該如何獲得幫助？

    4. 恩道電子書客服郵箱：ebookservice@endao.co

    恩道小管家微信號：Endao_books_ipcs

    如有任何未在此提及的問題需要解決，均可通過以上方式進行反饋，請盡量詳細描述您所遇到的問題，同工收到後將會盡快回復。


靈修功能相關

  1. 1. 如何使用靈修功能？

    2. 您可以在App「靈修>全部靈修」中選擇感興趣的靈修/讀經材料，點擊「添加至靈修計劃」後即可在「我的靈修」中在線接收每日推送的靈修/讀經內容，書寫感悟筆記，並在完成後打卡。

  2. 2. 為什麼我只能閱讀靈修計劃當日的內容？

    3. 未購書時，您僅可查看當日靈修內容，無法補讀或預覽其他內容；購書後則可暢讀所有內容，並能通過日曆功能跳轉到未打卡的日期，進行「補打卡」操作。

  3. 3. 我的靈修筆記可以導出嗎？

    4. 您可以在App「我的>筆記>靈修筆記」中，進入對應靈修材料的筆記詳情頁面，點擊右上角「導出」按鈕，批量導出您的靈修筆記。
